# Tellama

**Private local AI for Android. Built for people who want powerful models, local control, and a phone-first experience.**

[Privacy Policy](https://redpluglab.github.io/tellama/privacy/) · [Public Site](https://redpluglab.github.io/tellama/)

Tellama turns an Android device into a private local AI workspace. It can import or download supported GGUF models, run readiness checks before loading, chat locally, and expose an optional Ollama/OpenAI-compatible local API server for the user's own tools.

This public repository contains release pages and public product information. The commercial Android source code is maintained privately by redplug, Inc.

## Highlights

- Local GGUF model execution powered by llama.cpp.
- Phone-first model catalog with large-model filtering.
- Preflight checks for memory, storage, battery, and thermal risk.
- Local Only API server by default.
- Optional LAN mode with API key authentication and rate limiting.
- Encrypted local chat history and user-controlled Memory.
- Safe Mode diagnostics and redacted support reports.
- Optional voice input through the Android system speech service.
- Google Play release documents aligned with privacy, Data safety, and foreground service requirements.

## Privacy

Tellama does not require a Tellama account for local use. Prompts, responses, chat history, Memory items, API keys, and diagnostics are designed to stay on the user's device unless the user explicitly chooses otherwise.

Read the public privacy policy: https://redpluglab.github.io/tellama/privacy/
