# Tellama public roadmap

This roadmap covers public interfaces and integration work. It is directional and does not promise release dates.

## Available

- Authenticated Ollama-style model discovery, chat, generate, show, and process routes
- OpenAI-style model discovery and streaming chat completions
- Python and JavaScript clients with contract tests
- Wi-Fi LAN server controls with automatic stop on network loss

## Next

- Publish a reproducible Android device and model benchmark format
- Expand compatibility fixtures for desktop agents and local AI tools
- Add more client examples without exposing Android application internals
- Document tested model/device combinations and thermal behavior
- Improve request cancellation and concurrency reporting

## Under evaluation

- Embeddings API
- Tool-call compatibility
- Optional TLS through a trusted local reverse proxy
- Additional SDK languages based on demonstrated demand

The Tellama Android application source and native runtime remain private. Public work will focus on stable interfaces, clients, tests, and transparent compatibility data.
