# Tellama v1.1.6

Tellama v1.1.6 is a commercial-stability update for model storage, downloads, and accessibility.

## Improvements

- Resumes verified large-model downloads after app process termination.
- Keeps a 1 GB storage reserve and blocks unsafe redirects or oversized responses.
- Switches trusted mirrors after a checksum mismatch and rejects invalid GGUF files.
- Stops active generation and local API serving before deleting a loaded model.
- Releases native model mappings before reclaiming the model file and per-model settings.
- Improves large-text actions and scroll behavior in compact or landscape layouts.
- Verifies TalkBack focus, tablet-width layouts, and Arabic RTL behavior.
- Completes the newest chat, server, profile, and error copy across all 15 supported languages.

## Verification

- 112 release unit tests passed.
- Android Lint completed with 0 errors.
- APK 16 KB ZIP alignment and all arm64 ELF `0x4000` alignments passed.
- A 1.0 GB model resumed from 49% after process termination and completed verification.
- Loaded-model deletion reduced measured app RSS from about 1.44 GB to 329 MB and reclaimed about 1.09 GB of device storage.
