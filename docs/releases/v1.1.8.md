# Tellama v1.1.8

Tellama v1.1.8 adds a bounded, user-controlled web research workflow while preserving the product's local-first security model.

## New

- Reads up to three public HTTPS pages for a summary or comparison.
- Keeps source links with the generated answer so they remain available after restart.
- Requires explicit user approval before a web result is saved to model-independent long-term memory.
- Supports cancellation during page reading and local generation.

## Safety and reliability

- Blocks HTTP, local and private network targets, authentication paths, unsafe redirects, unsupported content types, and responses over 2 MB.
- Treats page text as untrusted reference material instead of model instructions.
- Warns before page text is sent to a configured external API model.
- Preserves encrypted chats and memories during the database upgrade.
- Improves non-streaming compatible API responses and runtime diagnostics.
- Applies the new Tellama goat launcher icon, Android themed icon, Play Store asset, and browser icon set.
