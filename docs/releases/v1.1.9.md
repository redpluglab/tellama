# Tellama v1.1.9

Tellama v1.1.9 is a controlled tester release focused on Gemma 4 stability, trustworthy device status, resilient model handling, and less disruptive updates.

## Highlights

- Hardens Gemma 4 E2B prompt formatting, thinking-output cleanup, model unloading, and conservative mobile thread use.
- Keeps workspace conversation, memory, speed, and device metrics synchronized with persisted activity.
- Improves selected and serving model states, recommendation badges, resumable downloads, deletion, and GGUF import feedback.
- Preserves global long-term memory when switching models.
- Checks for new releases at startup no more than once every six hours and shows the result inside Tellama.
- Suppresses a dismissed version for 24 hours; update-check failures never block local use.

## Verification

- 135 unit tests passed; Android Lint completed with 0 errors.
- R8 release packaging and APK Signature Scheme v3 verification passed.
- Signed v1.1.8 to v1.1.9 upgrade completed on a physical Android 10 device with existing chats, memory, and model selection retained.
- Cold start completed without fatal exception, ANR, JNI, or out-of-memory records.
