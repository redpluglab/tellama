# Tellama v1.2.10

Tellama v1.2.10 improves chat usability and lands a broad commercial-release hardening pass (native runtime, prompt budgeting, streaming, server API, and model storage).

## Highlights

- **Long-press message actions.** Long-pressing a chat message now opens a context menu — Copy, Save to memory, Delete — instead of jumping straight to memory-save. Copy writes the message to the clipboard. Localized across all 15 languages.
- **Cleaner Tellama badge.** The "thinking" indicator shown while a reply is generated now renders the Tellama mascot as a clean circular badge on the launcher mint background, consistent with the app icon.
- **More reliable long conversations.** Fixed a context-budget miscount that could make some models (Gemma family) fail an ordinary multi-turn chat.
- **Better text handling.** Emoji and other non-BMP characters are now tokenized correctly.
- **Smoother, safer generation.** Streaming updates are throttled to a frame interval (less jank), generation cancel is reported accurately, and the KV cache stays consistent on recurrent/hybrid models.
- **OpenAI-compatible server.** Responses now include the `id`/`created` fields and a terminal `finish_reason` chunk that official OpenAI SDKs require.
- **Storage & download hardening.** Continued STORE_002 race hardening around model download, delete, and server handoff.
