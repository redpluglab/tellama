# Tellama v1.2.11

Tellama v1.2.11 prepares the first Google Play production submission with permission minimization and privacy-disclosure accuracy, on top of the v1.2.10 chat and hardening improvements.

## Highlights

- **Fewer permissions.** Removed `RECORD_AUDIO` and `ACCESS_WIFI_STATE`. Voice input uses the Android system speech recognizer, which owns the microphone, so the app itself no longer requests mic access.
- **Clearer external-API disclosure.** When you add an external OpenAI-compatible provider, the app now states at the point of configuration that your messages are transmitted to that provider.
- Includes the v1.2.10 additions: long-press message actions (Copy / Save to memory / Delete), the refreshed Tellama badge, long-conversation stability, and correct emoji handling.
