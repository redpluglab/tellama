# Tellama v1.2.3

Tellama v1.2.3 is a settings UX overhaul plus a production-hardening pass, from a full commercial-quality code review.

## Highlights

- Reorganized Settings into a large-tech-style hub with grouped sub-pages (Chat mode, Language, Theme, Text size, Model downloads, Memory, Safety & Diagnostics, Updates, About).
- Fixed 24 production defects surfaced in review — most notably no longer unloading the running model on `onTrimMemory` UI_HIDDEN/BACKGROUND; the model is now released only under real memory pressure and never while the local server is running.
- Hardened database recovery to rebuild the encrypted store on corruption instead of failing to open.

## Verification

- Clean unit-test executions across Direct/Play × Debug/Release passed with no failures.
- Signed APK passed package/version inspection, zip alignment, and APK Signature Scheme v2 certificate verification.
- Installed over the prior version on a physical device with conversations, memory, and installed models retained.
