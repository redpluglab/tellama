# Tellama v1.2.5

Tellama v1.2.5 rebuilds the first-run experience so new users always know what to do: a friendly goat mascot walks you through onboarding and sets everything up automatically.

## Highlights

- Character-guided onboarding — a three-page walkthrough narrated by the Tellama goat, with full-body character illustrations, localized in all 15 languages.
- Automatic setup — "Get started" inspects the phone, picks a model that fits it (respecting minimum/recommended RAM, skipping reasoning-only models), downloads it with live progress, and warms it into the runtime.
- A one-time bouncing arrow points at the Chat tab when setup finishes, where the model is already loaded and ready.
