# Tellama v1.2.6

Tellama v1.2.6 polishes three everyday screens based on real-device feedback: a roomier chat composer, a manual Safe Mode switch, and a clearer Server screen.

## Highlights

- Chat — a two-row composer: the message field now spans the full width, with attach/web/voice and send on a compact row beneath it.
- Settings — a manual Safe Mode switch in Safety & Diagnostics. Turning it on blocks the local API server and runs models with a reduced context window; the diagnostics gate honors the switch everywhere.
- Server — reordered to model → controls → connection info → API keys, with a "Who can connect" label and rewritten copy that says what actually happens.

## Verification

- Galaxy S10 (Android 12) upgrade install with data retained: two-row composer, Safe Mode switch on/off, reorganized Server screen.
- 724 unit-test executions passed across Direct/Play × Debug/Release variants.
