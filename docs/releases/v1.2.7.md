# Tellama v1.2.7

Tellama v1.2.7 adds an in-app link to the new illustrated user guide and ships alongside a refreshed homepage with a Docs section.

## Highlights

- Settings → About now has an "Open user guide" button that opens the step-by-step, hand-sketched manual on the Tellama site (installation, first-run setup, chatting, Safe Mode, and the local server).
- The guide URL is configurable via build config rather than hard-coded.
- Companion site update: a new Docs entry in the top navigation and an illustrated guide page, with homepage version and integrity details brought current.

## Verification

- Galaxy S10 (Android 12) upgrade install with data retained: the guide button fires an ACTION_VIEW intent to the guide URL and opens the browser.
- 724 unit-test executions passed across Direct/Play × Debug/Release variants.
