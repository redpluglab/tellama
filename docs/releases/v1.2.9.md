# Tellama v1.2.9

Hotfix for a crash introduced in v1.2.8 when the chat thinking indicator appeared after sending a message.

## What changed

- **Chat crash fix.** The animated thinking badge no longer loads the adaptive launcher mipmap through Compose `painterResource` (unsupported). It now uses the raster Tellama foreground drawable, so sending a chat message no longer crashes the app.
- All other v1.2.8 improvements are retained (multi-turn context, new models, smoother generation).

## Who should update

Everyone on **v1.2.8** should update immediately. If you are on v1.2.7 or earlier, updating to v1.2.9 is still recommended.
