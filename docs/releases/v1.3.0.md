# Tellama v1.3.0

Tellama v1.3.0 focuses on chat usability, verified on-device (Galaxy S10).

## Highlights

- **Newest message first on open.** Opening a conversation now jumps to the latest message instead of the oldest, so you no longer have to scroll all the way down.
- **Keyboard no longer covers the input.** The composer and send button rise above the on-screen keyboard while typing.
- **Model list grouped by family.** The catalog is organized into families (Qwen, Llama, Gemma, …) with each family sorted from the smaller to the larger model for easier scanning.
- Minor polish, including moving the web-source citation label into localized resources.
