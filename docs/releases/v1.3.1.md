# Tellama v1.3.1 — Gold Master

Tellama v1.3.1 restores on-device generation speed, verified on-device (Galaxy S10).

## Highlights

- **Generation is fast again.** On-device decode speed is back to normal — on a Galaxy S10 (Qwen2.5 1.5B Q4_K_M), decode recovered from ~0.3 to **9.6 tok/s** and prefill to **52 tok/s**.
- **No more overheating.** The same conversation that used to heat the device now stays cool (measured 31.9 °C battery).
- **Structural root-cause fix.** The native compute kernels (ggml/llama.cpp) were being compiled without optimization (-O0/-O2) because the Android build never triggered llama.cpp's own "force Release" path. They now always compile at **-O3**, and the generation thread is prioritized so the scheduler favors the fast (big) CPU cores.
