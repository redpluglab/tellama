# Tellama v1.3.3

Tellama v1.3.3 makes Qwen3 answer fast.

## Highlights

- **Qwen3 responds quickly now.** Qwen3 base models (like Qwen3 4B) are hybrid — they support both a thinking mode and a fast non-thinking mode. Tellama previously treated them as thinking-only and forced every reply through hidden chain-of-thought, so the first visible word was slow to appear. They now run in Fast mode by default (no hidden reasoning), so answers start right away. Switch to Deep Thinking in Settings when you want full reasoning.
- Genuinely reasoning-only models (QwQ, DeepSeek-R1, Qwen3 *Thinking* variants) still use Deep Thinking as before.
