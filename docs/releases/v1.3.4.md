# Tellama v1.3.4

Tellama v1.3.4 makes more models usable on your phone and keeps your question in view.

## Highlights

- **Accurate memory requirements.** The RAM figures shown for each model were overstated for 4-bit models — some by 2–3×. They are now derived from the model's actual file size, so capable phones see the models they can really run.
- **Your question stays visible.** After you send a message, the chat scrolls so your just-sent question is fully in view above the keyboard, instead of being cut off.
- **Small fix.** Phi 3.5 Mini no longer shows an inaccurate "reasoning-heavy" hint.
