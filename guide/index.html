Tellama, 이렇게 쓰면 됩니다

APK 설치부터 첫 대화, 안전 모드와 로컬 서버까지 — 각 단계마다 만나게 될 화면을 손그림 스케치와 함께 차근차근 안내합니다.

STEP 01

APK 설치하기

지인 테스트 동안 Tellama는 서명된 APK로 제공됩니다. 릴리즈 페이지에서 한 번만 설치하면 됩니다.

  1. 릴리즈 페이지를 열어 최신 tellama-v1-2-7.apk를 내려받습니다.
  2. 안드로이드가 물어보면 이번 한 번만 브라우저에 ‘알 수 없는 앱 설치’를 허용합니다.
  3. Play Protect 검사가 끝나면 설치를 누릅니다.

STEP 02

첫 실행 — 산양이 반겨줘요

앱을 처음 열면 친근한 3장짜리 안내가 Tellama를 소개합니다.

  1. 산양 캐릭터가 안내하는 3장의 소개 페이지를 넘겨봅니다.
  2. 무엇이 기기에 남고, 모델 다운로드에는 무엇이 필요한지 확인합니다.
  3. ‘시작하기’를 누르면 자동 설정이 시작됩니다. 모든 단계는 건너뛸 수 있어요.

STEP 03

자동 설정

Tellama가 기기를 살펴보고, 사용자가 아무것도 고르지 않아도 알맞은 모델을 준비합니다.

  1. RAM과 저장공간을 확인해 기기에 맞는 모델을 자동으로 고릅니다.
  2. 진행률 막대와 함께 모델을 내려받고 메모리로 예열합니다.
  3. ‘준비 완료!’가 뜨면 통통 튀는 화살표가 채팅 탭을 가리킵니다.

STEP 04

첫 대화 시작하기

채팅 탭을 열면 모델이 이미 로드되어 바로 대화할 수 있습니다 — 완전히 오프라인으로요.

  1. 아래쪽 전체 폭 입력창에 메시지를 씁니다.
  2. 바로 아래 도구 줄에서 파일·웹·음성을 붙일 수 있습니다.
  3. 전송을 누르면 응답이 실시간으로 흐르고, 메모리에 저장할 내용은 직접 승인합니다.

STEP 05

모델 고르기·바꾸기

모델 탭에서는 기기 여유에 맞춰 가벼운 일상 대화부터 더 깊은 추론 모델까지 옮겨갈 수 있습니다.

  1. 각 모델에는 RAM 배지와 준비 상태가 표시됩니다.
  2. 기기에 맞는 모델을 내려받아 서빙 모델로 지정합니다.
  3. 전환은 안전합니다 — 새 모델을 올리기 전에 기존 모델을 먼저 내립니다.

STEP 06

불안하면 안전 모드

앱이 불안정하게 느껴질 때, 안전 모드는 모든 것을 가장 보수적인 상태로 실행합니다.

  1. 설정 → 안전 모드 및 진단을 엽니다.
  2. ‘안전 모드 직접 켜기’ 스위치를 켭니다.
  3. 끄기 전까지 로컬 서버 시작이 막히고 모델은 축소된 컨텍스트로 실행됩니다.

STEP 07

다른 기기에서 쓰기

서버 탭은 내 폰을 신뢰한 기기용 로컬 Ollama/OpenAI 호환 AI 서버로 만듭니다.

  1. 접속 허용 범위를 고릅니다: 이 휴대폰만, 또는 Wi-Fi.
  2. Wi-Fi 접속은 API 키를 먼저 만든 뒤 서버를 시작합니다.
  3. 주소나 준비된 curl 예제를 복사해 다른 기기의 앱에 붙여 넣습니다.
