Private local AI for Android

Your phone, your model, your AI workspace.

Tellama brings local GGUF model execution, private chat, user-controlled Memory, device safety checks, and an optional Ollama/OpenAI-compatible API server into one refined Android app.

Privacy Policy GitHub Repository

Local by design

Prompts, responses, chat history, Memory items, API keys, and diagnostics are designed to stay on device unless the user explicitly chooses otherwise.

Phone-first runtime

Tellama focuses on practical Android deployment with model size filtering, readiness checks, and safety states built for mobile constraints.

Developer-ready

The optional local API server supports familiar Ollama/OpenAI-style workflows for the user's own local tools.

Built for serious local AI

  • Local GGUF model execution powered by llama.cpp.
  • Phone-first model catalog with large-model filtering.
  • Preflight checks for memory, storage, battery, and thermal risk.
  • Local Only API server by default, with optional LAN mode.
  • API key authentication and rate limiting for serving workflows.
  • Encrypted local chat history and user-controlled Memory.
  • Safe Mode diagnostics and redacted support reports.
  • Optional voice input through the Android system speech service.
