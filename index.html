인터넷이 항상 필요한가요?

설치된 로컬 모델과의 채팅은 오프라인으로 사용할 수 있습니다. 모델·앱 다운로드, 웹 리서치, 외부 API 모델은 네트워크가 필요합니다.

앱 업데이트는 안전한가요?

v1.2.1 이상은 APK의 SHA-256, 패키지명, 상위 버전과 서명 인증서를 검증한 뒤 Android 설치 화면을 엽니다.

외부 API 모델도 개인정보가 로컬에만 남나요?

아닙니다. 외부 API 모델을 선택하면 입력과 필요한 문맥이 사용자가 구성한 제공자에게 전송되며 해당 제공자의 정책과 과금 조건이 적용됩니다.
