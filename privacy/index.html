Tellama Privacy Policy

Effective date: July 8, 2026

Tellama is a local AI app for Android. It lets you import or download supported local model files, chat with models on your device, save user-approved Memory items, and optionally run a local API server.

This policy applies to the Tellama Android app published as com.tellama.app.

Data We Store on Your Device

Tellama stores chat sessions, prompts, responses, user-approved Memory items, installed or imported model metadata and checksums, local API client records, failure records, diagnostics, and user settings locally on your Android device.

API keys are stored as SHA-256 hashes; the raw key is shown only once. Tellama uses an encrypted local database. Android backup is disabled for app data.

Voice Input

Tellama requests microphone permission only when you use voice input. Voice recognition is handled by the Android system speech service installed on your device. Tellama does not store raw audio recordings. Recognized text is stored only if you send it as a chat message.

Network Use

Tellama does not require an account for local use. Tellama uses network access for user-initiated model downloads from the selected model provider over HTTPS and for optional local API serving.

The API server is off by default. Local Only mode binds to 127.0.0.1. LAN mode requires explicit user action and API key authentication. Local LAN API traffic is not TLS-encrypted, so enable LAN mode only on networks you trust.

Tellama does not send prompts, responses, chat history, Memory items, API keys, or voice transcripts to Tellama-controlled cloud servers.

Support Reports

Support reports are generated only when you choose to view or share them. Tellama redacts bearer tokens, API keys, and local filesystem paths before presenting support data.

Data Sharing

Tellama does not sell user data and does not share user data with advertising networks.

Data may leave the app only when you explicitly share a redacted support report, download a model file from a third-party model host, use Android system speech recognition, or enable local API serving and connect your own local clients to the app.

Data Deletion

You can delete local Tellama data by resetting Memory in Tellama Settings, clearing Tellama storage in Android system settings, or uninstalling Tellama. Because Tellama does not require a Tellama account or run a Tellama cloud account service, there is no server-side account data to delete.

Children

Tellama is intended for developers, local AI users, and general productivity use. It is not directed to children.

Changes

If this policy changes, the updated version will be published at https://redplug.github.io/tellama/privacy/.

Contact

For privacy questions, contact support@tellama.app.
