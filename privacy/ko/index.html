개인정보 처리방침

Tellama는 안드로이드에서 로컬 AI를 실행하도록 설계되었습니다. 사용자가 연결 기능을 직접 선택하지 않는 한 대화, 응답, 대화 기록, 메모리, API 키와 진단 정보는 기기 안에 보관됩니다.

시행일 2026년 7월 13일패키지 com.tellama.app계정 불필요
핵심 요약: 로컬 모델의 프롬프트와 응답은 Tellama가 관리하는 클라우드 서버로 전송되지 않습니다. 외부 API 모델, 웹 리서치, 모델·앱 다운로드, Android 음성 인식, 지원 보고서 공유와 로컬 API 서빙은 사용자의 선택에 따라 네트워크 전송 또는 공유가 발생할 수 있으며 해당 제3자의 약관이 적용됩니다.

기기에 저장하는 정보

Tellama는 대화 세션, 프롬프트, 응답, 사용자가 승인한 메모리, 설치·가져오기한 모델의 메타데이터와 체크섬, 로컬 API 클라이언트·실패 기록, 진단 정보, 설정과 선택적 프로필 정보를 Android 기기에 저장합니다.

로컬 API 서버용 클라이언트 키는 SHA-256 해시로 저장하며 원문은 생성 시 한 번만 표시합니다. 외부 제공업체용 키는 해당 업체 인증을 위해 암호화된 로컬 데이터베이스에 저장합니다. 로컬 데이터베이스는 암호화하며 Android 앱 데이터 백업은 비활성화합니다.

선택적 프로필과 제품 소식

로컬 사용에는 계정이 필요하지 않습니다. 설정에서 이름, 이메일, 프로필 사진을 선택적으로 추가할 수 있으며 사용자가 공유하지 않는 한 기기 안에만 보관됩니다.

선택적 프로필(이름·이메일·사진)과 "제품 소식" 설정은 Tellama의 기기 내 비공개 저장소에만 보관됩니다. Tellama는 프로필이나 이메일 주소를 외부로 전송하지 않으며 계정도 없습니다. "제품 소식" 토글은 로컬 설정일 뿐입니다.

음성 입력

Tellama는 마이크 권한을 요청하지 않습니다. 음성 입력은 Android 시스템 음성 인식기를 실행하며, 마이크는 시스템 음성 서비스가 직접 소유·처리하고 Tellama에는 인식된 문장만 전달됩니다. 원본 음성은 저장하지 않으며, 인식된 문장은 사용자가 채팅으로 전송했을 때만 대화에 저장됩니다.

네트워크 사용

사용자가 시작한 모델 다운로드, GitHub 릴리즈 확인과 앱 업데이트, 외부 API 모델, 사용자가 지정한 공개 HTTPS 페이지에 대한 웹 리서치, Android 음성 서비스 및 선택적 로컬 API 서빙에 네트워크를 사용합니다.

API 서버는 기본적으로 꺼져 있습니다. 로컬 전용 모드는 127.0.0.1에 바인딩합니다. Wi-Fi LAN 모드는 사용자가 직접 켜야 하며 API 키 인증이 필요합니다. LAN 트래픽은 TLS로 암호화되지 않으므로 신뢰하는 네트워크에서만 사용해야 합니다.

외부 API 모델

OpenAI 호환 외부 API 모델을 추가하거나 선택하면 요청에 필요한 프롬프트와 문맥을 사용자가 설정한 엔드포인트로 전송합니다. 해당 제공자의 개인정보, 보유, 보안, 콘텐츠 및 과금 조건이 적용됩니다. Tellama는 외부 모델 사용 전에 앱 안에서 이를 안내합니다.

웹 리서치

사용자가 입력한 공개 HTTPS URL에 직접 요청합니다. 로컬·사설 네트워크 대상, 안전하지 않은 리디렉션, 인증 경로, 지원하지 않는 형식과 과도하게 큰 응답을 차단합니다. 로컬 모델을 사용하면 페이지 텍스트를 기기에서 처리하지만, 외부 API 모델로 요약·비교하면 앱 안의 사전 경고 후 필요한 페이지 내용을 해당 제공자에게 전송합니다.

모델 다운로드와 앱 업데이트

모델 다운로드는 사용자가 선택한 제3자 모델 제공처에 연결하며, 제공처는 IP 주소와 사용자 에이전트 같은 일반적인 요청 정보를 받을 수 있습니다. Tellama는 지원하는 모델 파일을 사용하기 전에 검증합니다.

Tellama는 공개 GitHub 릴리즈 피드에서 새 버전을 확인할 수 있습니다. 업데이트를 시작하면 GitHub에서 APK를 받고 SHA-256, 패키지명, 상위 버전과 서명 인증서를 검증한 뒤 Android 설치 화면을 엽니다.

지원 보고서

사용자가 보기 또는 공유를 선택했을 때만 생성합니다. 표시 전에 Bearer 토큰, API 키와 로컬 파일 경로를 가립니다.

정보 공유

Tellama는 사용자 정보를 판매하거나 광고 네트워크와 공유하지 않습니다.

외부 API 모델, 웹 리서치, 모델·앱 다운로드, 지원 보고서 공유, 원격 처리를 사용하는 Android 음성 서비스 또는 사용자가 켠 로컬 API 클라이언트 연결을 통해 정보가 앱 밖으로 나갈 수 있습니다.

보유 및 삭제

로컬 정보는 사용자가 삭제하거나 Tellama 저장공간을 지우거나 앱을 제거할 때까지 기기에 남습니다. 프로필 편집, 설정의 메모리 초기화, Android 앱 저장공간 삭제 또는 앱 제거로 로컬 정보를 삭제할 수 있습니다.

Tellama는 외부 API 제공자, GitHub, 모델 제공처, 웹사이트 운영자, 이메일 제공자 또는 Android 음성 서비스의 보유 정책을 통제하지 않습니다. 해당 기능을 사용하기 전에 제공자의 정책을 확인해야 합니다.

아동

Tellama는 개발자, 로컬 AI 사용자와 일반 생산성 사용자를 위한 앱이며 아동을 대상으로 하지 않습니다.
